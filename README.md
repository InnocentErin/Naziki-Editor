# Naziki Editor
## 针对Cytoid故事板设计的可视化编辑器！  

  该项目并未完成，请耐心等候。

  该项目的立项旨在为Cytoid游戏玩家/谱师提供一个更加方便的故事板设计环境。  
  让故事板设计师们摆脱重复编辑源代码的低效设计过程，专注于设计精美故事板这件事情本身。  

----------------------------------------------------------------------------

## 项目相关
  语言：Csharp  
  IDE：Visual Studio  
  辅助：Google Gemini  

----------------------------------------------------------------------------

## 需要注意：  
  该项目并未测试各个功能的可用性和兼容性。  
  该项目使用AI辅助，由Erin分模块搭建。

  欢迎各位设计师前来体验~  
  各位设计师如果遇到了bug，可以提交issue反馈，  
  虽然我不懂代码，不保证能够持续更新，  
  但我还是会尽力去完善相关功能！  

  如果可以的话，
  拜托各位代码大佬尽力优化这个项目！  
  不懂代码的Erin在这里谢谢各位了！

## Erin的笔记！  
本人对Storyboard甚至是代码一窍不通，  
制作的第一个storyboard level还是和其他人合作的。  
那个合作的人现在早已退游……（所以这个项目以他命名）  
所以我的整个关卡制作生涯中没有几个涉及故事板的关卡。  

老实说，成立这个项目确实有纪念他的因素。  
可是我不能因为失去了他而不去涉足故事板设计，这个游戏最为精华的部分。  

本来这个项目是不会存在的，我只需要试着找其他人与我一起合作完成故事板关卡的。  
可惜周围了解storyboard的人们基本上都在忙，没有时间帮助我。  
我在思考，连可视化谱面编辑器都被设计出来了，为什么没有人设计storyboard可视化编辑器呢。  
趁着现在的AI风口，我尝试踏出第一步。  

我知道这个由AI辅助搭建的Cytoid Storyboard Editor，
易用性和维护性相对正常开发的项目可维护性会差一些，  
但是还是希望至少能够通过它，让故事板的设计者们减少一些制作故事板的门槛。  
希望有人能帮助修补这个项目，让它能够迸发出更强大的能量，  
也希望有人能够制作出一个比这个更好用的Cytoid Storyboard Editor！
